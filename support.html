Support

Select Your Setup

Follow tailored troubleshooting steps based on your setup.

Windows Support

Troubleshooting Step 1

  • Ensure CrossyBot has been stopped
  • Download CrossyBot (top right) and extract to a safe location
  • Copy and paste model.db file into the extracted folder next to crossybot_windows.exe
  • Start CrossyBot

If your problem still has not been fixed:

Troubleshooting Step 2

  • Ensure CrossyBot has been stopped
  • Delete the model.db file
  • Restart CrossyBot
⚠️ Warning: All CrossyBot settings and config will be lost. You will need to set up CrossyBot again by entering your valid Telegram bot token and API keys. Your credits will not be impacted by this action.

AWS Support

Troubleshooting Step 1

  • Download CrossyBot (top right) and extract to a safe location
  • Copy and paste your AWS .pem file into the extracted folder
  • Double click on the crossybot_aws_helper.bat file to launch it
  • Ensure CrossyBot has been stopped
  • Complete:
    • Migrate model.db from AWS to Windows
    • Uninstall
    • Install
    • Migrate model.db from Windows to AWS
    • Start bot

If your problem still has not been fixed:

Troubleshooting Step 2

    • Double click on the crossybot_aws_helper.bat file to launch it
    • Ensure CrossyBot has been stopped
    • Complete uninstall
    • Complete install
    • Start bot
    ⚠️ Warning: All CrossyBot settings and config will be lost. You will need to set up CrossyBot again by entering your valid Telegram bot token and API keys. Your credits will not be impacted by this action. The easiest way to set up a new model.db file is by double clicking the crossybot_windows.exe file, completing login, then using the helper script to migrate the new model.db to AWS.
