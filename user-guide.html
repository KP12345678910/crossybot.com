üì± How Telegram Works

  • Telegram bots function through commands starting with '/', button clicks or messages sent into the chat when prompted
  • Bots are workflow-Based, requiring you to move in and out of menus
  • Send /start any time to show the main menu
  • Always interact with the latest message from the bot

‚öôÔ∏è How the Execution Engine Works

  • Updates open trades every 15 seconds minimum
  • Executes trades based on your balances, automations, and excluded pairs
  • Excluded pairs will not trigger automations or trade stop losses (but equity stop losses still apply)
  • Pause all automations at any time using the /pause command
  • At first startup, CrossyBot excludes any open balances‚Äîyou must remove them manually

üö® IMPORTANT: CrossyBot only supports USD pairs.
It‚Äôs best to be in all USD balance before starting. Always check menu data before removing exclusions.

Quicktrade allows you to execute long, spot or short trades with no CrossyBot fees.
Using quicktrade aims to ensure that maximum margin allocations &/or risky positions against the trend's direction are not taken unless 100% certain.

You will need to select or enter in your pair using the chat menu to proceed. Please note that pairs are not always standard (i.e. XXBT = BTC) so please check the explore pairs menu when pairs are not found.

Quicktrade requires you select a % of your portfolio to allocate to the trade. The selection options will be reduced depending on how much available &/or collatoralised usd you hold in margin positions.

Quicktrade requires you to confirm any actions before completed at the exchange. Again, no fees will be charged.

Prices and EMAs lets you visualise what CrossyBot is doing on the go.

Select a pair, send a pair into chat or click an open balance charts button if you have open balances to view.
‚Ä¢ Show all open balances sends both the higher timeframe and lower timeframe charts into the chat
‚Ä¢ Open HTFs sends the higher timeframe charts for all open balances into the chat
‚Ä¢ Open LTFs sends the lower timeframe charts for all open balances into the chat

Charts can be deleted from the chat with the delete button.
You can also view charts at any time in CrossyBot by using the /charts command.

CrossyBot's settings are crucial to the way it behaves. Settings are split out into 3 submenus:
‚Ä¢ Automation settings
‚Ä¢ Trade settings
‚Ä¢ Portfolio settings

There is an additional button which can be used to update API keys should you wish to.

Automation settings sit at the crux of Crossybot operation.
‚Ä¢ Pause automations ‚Äì lets you pause all crossybot automations
‚Ä¢ Exclude automations ‚Äì lets you exclude specific pairs from all crossybot automations
‚Ä¢ Set ema ranges ‚Äì key settings for crossybot moving average signal generation
‚Ä¢ Set timeframes ‚Äì key settings for crossybot moving average signal generation
‚Ä¢ Max automation value ‚Äì a hard cap to the value of automations crossybot will execute
‚Ä¢ Htf notifications ‚Äì enable/disable notifications for higher timeframe crossovers when you have open balances or positions

Tweaking EMA ranges and timeframes helps balance precision and signal quality. Narrow settings increase noise and frequent signals, while broader ones smooth the trend and widen the spacing between crossovers.

Trade settings let you apply finer control over how CrossyBot operates:
‚Ä¢ Taker fee rate ‚Äì the fee rate set by Kraken used in CrossyBot PnL calculations
‚Ä¢ Profitable close % ‚Äì % threshold at which open trades are considered profitable and can be auto-Closed
‚Ä¢ Trade stop loss ‚Äì % drop from entry where CrossyBot will automatically close positions (can be updated live)

Portfolio settings help look after the bottom line:
‚Ä¢ Equity stop loss ‚Äì sets a % portfolio loss threshold to sell all balances and stop automations
‚Ä¢ Max margin allocation ‚Äì sets max % of portfolio used for margin collateral
CrossyBot uses 2x leverage by default, so 50% max margin is the default. If using higher leverage manually, consider reducing this to e.g. 100% √∑ leverage.
