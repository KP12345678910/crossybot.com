2. Description of Service

CrossyBot is an execution-only cryptocurrency trading automation tool. It enables users to automate trade execution based on user-defined parameters such as EMA crossover strategies. The Software integrates with the Kraken API using user-provided API keys and offers a Telegram-based interface for interaction and control.

CrossyBot is strictly a tool for executing user-directed actions. It does not provide financial, investment, or trading advice, and should not be relied upon as such. Users retain full control and responsibility over all trading decisions and parameters.

The client application is self-hosted by users, either on their own Windows device or optionally on a cloud-hosted EC2 instance. The CrossyBot API backend is hosted on a third-party platform and facilitates necessary supporting functions (e.g., account management, credit accounting, email notifications), but does not initiate trades or manage user funds.
