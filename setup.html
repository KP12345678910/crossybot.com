Setup Once, Run Forever

Setup CrossyBot to run locally or on AWS for 24/7 uptime.

1. Create Kraken API Keys

▼
  • Click account icon (top right)
  • Click Settings → API
  • Set permissions and name
  • Save both keys securely
(more info)

2. Create Telegram Bot Token

▼
  • Download Telegram app
  • Search for BotFather → /newbot
  • Name your bot → save token
  • Open your bot chat
(more info)

3. Run CrossyBot

▼
  • Download CrossyBot using the button top right
  • Extract contents to a folder in a safe location
  • Double click the crossybot_windows.exe
  • Enter a username (email address) & password
  • You may be asked for a security code if signing up or logging in from a new location (sent to your email)
  • When setting up for the first time you will be asked to:
    • - Enter your Telegram bot token
    • - Enter your Kraken API key
    • - Enter your Kraken API secret (API permissions will be checked)
    • - Enter your strategy settings (i.e. higher/lower timeframes & EMA ranges)

✅ You've now completed setup and can access your bot from Telegram using /start! 🎉

⚠️ Closed orders will be updated on first start — this may take some time. Please allow it to complete before operating CrossyBot.

🖥️🔒 Closing this window or your computer will stop CrossyBot from running. Look at the AWS setup guide for instructions on how to run CrossyBot 24/7

2. Create Telegram Bot Token

3. Create AWS Account

▼
  • Visit aws.amazon.com
  • Create an AWS account with your email address
  • Verify phone number
  • Wait for confirmation email (can take up to 24 hours)
(more info)

4. Launch EC2 Instance

▼
  • Go to AWS Console → Search EC2
  • Ensure you are creating in your region (top right)
  • Launch Amazon Linux t2.micro
  • Create key pair (.pem) → download securely
  • Note Public IPv4 address
(more info)

5. Setup CrossyBot

▼
  • Download CrossyBot using the button top right
  • Extract contents to a folder in a safe location
  • First copy & paste your '.pem' file from AWS setup into the CrossyBot's 'aws' folder
  • Double click the crossybot_aws.bat in the main CrossyBot folder
  • Enter your AWS public IP address and hit enter
  • On successful connect you will see the mission control panel
  • Options are available depending on your extracted folder and AWS contents:
    • - Install - copies CrossyBot onto AWS
    • - Migrate model.db from windows - copies CrossyBot & model.db onto AWS
    • - Migrate model.db from aws - copies CrossyBot & model.db onto AWS
    • - Uninstall - removes CrossyBot & model.db from AWS
    • - Start bot - starts CrossyBot using powershell
    • - Stop Bot - stops CrossyBot
    • - Resume bot - resumes running bot using powershell
  • Complete install & start bot to start CrossyBot.

🧩 For the easiest AWS setup:
Setup model.db by logging in using:
crossybot_windows.exe > open crossybot_aws.bat > Install on AWS > Migrate model.db from windows > Start Bot
This ensures that model.db contains your Telegram bot token & API keys before being moved to AWS.

6. Run CrossyBot

▼
  • Double click the crossybot_aws.bat
  • Click on start bot or resume depending on if already running
  • Enter a username (email address) & password
  • You may be asked for a security code if signing up or logging in from a new location (sent to your email)
  • When setting up for the first time you will be asked to:
    • - Enter your Telegram bot token
    • - Enter your Kraken API key
    • - Enter your Kraken API secret (API permissions will be checked)
    • - Enter your strategy settings (i.e. higher/lower timeframes & EMA ranges)

✅ You've now completed setup and can access your bot from Telegram using /start! 🎉

⚠️ Closed orders will be updated on first start — this may take some time. Please allow it to complete before operating CrossyBot.

🖥️🔒 Closing crossybot_aws will not stop CrossyBot from running
